# Scaleflex Plugins

Landing page for Scaleflex plugins — digital asset management tools for the modern web.

**Live site:** https://scaleflex.github.io/scaleflex-pages/

## Plugins

| Plugin | Description | Links |
|--------|-------------|-------|
| **@scaleflex/asset-picker** | Browse and select assets from your Scaleflex DAM | [Demo](https://scaleflex.github.io/asset-picker/) · [Docs](https://scaleflex.github.io/asset-picker/#/docs/getting-started) · [GitHub](https://github.com/scaleflex/asset-picker) |
| **@scaleflex/uploader** | Upload files directly to your Scaleflex account | [Demo](https://scaleflex.github.io/uploader/) · [Docs](https://scaleflex.github.io/uploader/#/docs/getting-started) · [GitHub](https://github.com/scaleflex/uploader) |
| **@scaleflex/experience-picker** | Select and embed rich media experiences | [Docs](https://spotlight.scaleflex.com/docs/experience-picker) |

## Development

```bash
npm start
```
