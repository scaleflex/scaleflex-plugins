## Why Scaleflex Plugins?

- **Easy integration** — Drop-in widgets that connect to your Scaleflex account in minutes.
- **Lightweight** — Optimized builds for fast loading on any device and connection.
- **Accessible** — Keyboard navigation, screen reader support, and focus management.
- **Framework agnostic** — Works with React, Vue, Angular, Svelte, or plain HTML.
- **TypeScript support** — Full type definitions included out of the box.
- **Scaleflex DAM** — Native integration with [Scaleflex](https://www.scaleflex.com) for seamless asset workflows and global delivery.

## Running the Hub Locally

```bash
# Clone the repo
git clone https://github.com/scaleflex/scaleflex-plugins.git
cd scaleflex-plugins

# Start
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
